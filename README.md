# Pride Flags

A website to filter pride flags based on colors and patterns.

Most pride flags are sourced from Wikipedia.

## Stack

- [Node.js](https://nodejs.org/)
- [Eleventy](https://www.11ty.dev/)
- [Nunjucks](https://mozilla.github.io/nunjucks/)
- Vanilla CSS
- Vanilla JS

## Contributing

### Setup

```sh
npm install
```

### Development

```sh
npm run dev:site
```

### Linting

```sh
npm run lint:css
npm run lint:js
```

### Production

```sh
npm run build:site
```
