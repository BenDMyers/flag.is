# Pride Flags

A website to filter pride flags based on colors and patterns, available at [flag.is](https://flag.is).

## Stack

- [Node.js](https://nodejs.org/)
- [Eleventy](https://www.11ty.dev/)
- [Nunjucks](https://mozilla.github.io/nunjucks/)
- Vanilla CSS
- Vanilla JS

## Contributing

### Setup

```sh
npm install
```

### Development

```sh
npm run dev
```

### Formatting

```sh
npm run format
```

### Linting

```sh
# Run all linters
npm run lint

# Run individual linters
npm run lint:css
npm run lint:js
npm run lint:format
```

### Production

```sh
npm run build
```
